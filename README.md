# Advanced Computer Simulation and Modelling

**Course**: Advanced Computer Simulations and Modelling, SoSe 2026

**Instructor**: Prof. Dr. R. Steinigeweg, Universität Osnabrück

## About

> Does spin transport in the XXZ Heisenberg chain diffuse like ink in water, or flow ballistically like a frictionless pipe?

This repository includes notes/exercise code/project code for Steinigeweg's course on exact diagonalization and dynamical quantum typicality applied to quantum spin chains.

## Structure

| Folder | Contents |
|---|---|
| `notes/` | Annotated lecture notes |
| `exercises/` | Exercise notebooks |
| `projects/` | Final project code and writeup |
| `sandbox/` | Exploratory implementations |
| `reports/` | Submitted work reports |

## Technical

- **Language**: Mathematica 13.1
- **Topics**: Exact diagonalization, spin-1/2 XXZ chain, 
  symmetry reduction, dynamical quantum typicality
- **Hardware context**: Classical simulation; 
  NISQ comparison via prior PennyLane work
